Array to JSON with indexes as keys üîó

If you don‚Äôt want the direct string representation of a JSON array, you might want an object where the keys are the indexes of the array.

["apple", "orange", "banana"]

// becomes

{
  "0": "apple",
  "1": "orange",
  "2": "banana"
}

To get a JSON object from an array with index keys you can use the Object.assign method in conjunction with JSON.stringify.

const array = ["apple", "orange", "banana"];
const jsonString = JSON.stringify(Object.assign({}, array));
// jsonString becomes
// {"0":"apple","1":"orange","2":"banana"}
